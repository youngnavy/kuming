域名到底有多重要？致在路上的互联网创业者

发表于 2017-10-12 | 分类于 展示

何为域名？

举两个例子大家就明白了，百度用的网址是www.baidu.com，淘宝网用的网址是www.taobao.com，如果一个人需要到百度搜索内容或者到淘宝网购物，是不是直接输入这两个网址就可以了，就简单，最直接。

所以，baidu.com和taobao.com就是域名。

当然，域名有很多后缀，比如大家熟知的 .com / .net / .cn / .com.cn 还有其他新顶级后缀 .cc / .vip / .wang / .me 等等，举个例子，58.com是一个网址，可以建立网站，58.net / 58.vip 等等也是一个网站，可以建立网站。当然，有的后缀可以在国内备案，有的则不行，建议大家考虑的时候选择已经在国内通过备案的域名。

如何拥有自己的域名？

（1）注册

可以到万网（www.net.cn），西部数码（www.west.cn），易名中国（www.ename.cn）等等这些注册商注册，域名具有唯一性，如果想注册的域名，还没有被别人注册掉，那么就可以注册，费用大概一年几十元左右。

（2）购买

由于域名具有唯一性，短的，含义好的，基本早十年就被注册完了，如果自己喜欢的域名被注册了，那么只能通过查询域名持有人的联系方式，进行协商交易。

因此出现了，域名从业者，有的人专门注册很多短的，含义好的域名，然后出售给需要的一些人，因此有很多好的域名，可能在同一个人手中。因此域名也成了很多投资者的热爱。有的域名售出的价格甚至6位数，7位数，8位数。

域名到底重不重要？

例子太多，举一个简单的，当时很火的开心网，用的域名是（kaixin001.com)，但是用户想进入这个网站的时候，很多人不自觉的输入了（kaixin.com），损失了大量用户，如果（kaixin.com）建立一个和开心网类似内容的网站，那就是行业竞争对手了，那么开心网做越大，岂不是在为（kaixin.com)免费打广告？

小米八位数购买mi.com，从xiaomi.com升级为mi.com

京东八位数购买购买jd.com，从360buy.com升级为jd.com

360花费1个亿从国外拿下360.com

新浪微博花费800万购买weibo.com

等等案例，大公司都是非常重视域名保护的

致在路上的互联网创业者

互联网创业者越来越多，越来越难，百分之八十都需要一个域名，或用来做公司官方网站，或用来建立电商网站、交友网站等等。域名的重要性已经不言而喻了。

创业者们在选择域名的时候应该量力而行，但也不可随意选择。
